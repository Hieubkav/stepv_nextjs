Chào mừng bạn đến với tài liệu dự án. Nội dung tài liệu được viết bằng tiếng Việt, tập trung vào cách làm việc thực tế trong codebase hiện tại (Next.js + shadcn/ui + Tailwind v4 + Convex).

## Bắt đầu

- Xem nhanh kiến trúc: `apps/web` (frontend), `packages/backend` (Convex backend)
- Biến môi trường Convex: `apps/web/.env` (đang trỏ Convex cloud), có thể chạy local bằng `bun run dev:server`
